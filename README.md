multikanban
===========

A simple kanban for multiple personal projects.

#Instructions

    git clone git@github.com:mezod/multikanban.git
    npm install
    bower install
    
    if node was installed using homebrew then you might need to:
    npm install -g grunt-cli
  

#Techs

    npm
    bower
    grunt
    requirejs
    backbone
    marionette
    underscore
    bootstrap
    jquery
